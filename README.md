# snackonai
Hungry for smarter AI insights? SnackonAI delivers curated, bite-size AI news, tutorials, and tools in a weekly newsletter — no noise, just what matters.

## Projects

- **[mr-pixel-smith](./mr-pixel-smith)** — AI-powered image generation tool that creates watermarked images from text prompts using Ollama.
- **[snackonai-clips](./snackonai-clips)** — Turn any blog post URL into a LinkedIn-ready 1080×1920 vertical MP4 in seconds. Extracts article content, summarizes with OpenAI or Ollama, adds TTS voiceover (gTTS or ElevenLabs), and renders stylized video slides via MoviePy.
